# Job Portal Web Application

A full-stack Job Portal application with role-based access for Job Seekers and Employers.

## Features
### Authentication
- User Registration & Login (JWT based)
- Role-based access (Seeker / Employer)

### Job Seeker
- Profile Management
- Browse job listings
- Apply to jobs
- Track applied jobs

### Employer
- Create job listings
- View posted jobs
- View applicants per job

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Authentication: JWT
- Database: In-memory / JSON-based (can be extended to MongoDB)
